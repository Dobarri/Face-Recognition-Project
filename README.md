# Face-Recognition-Project

얼굴인식 프로그램을 직접 구현하려고 합니다.

- basic.py : 저장된 이미지영상에서 얼굴 인식
- AttendanceProject.py : webcam에서 실시간으로 영상에서 얼굴을 인식
- Attendance.csv : webcam에서 실시간 인식한 정보를 저장(Name, Time)
